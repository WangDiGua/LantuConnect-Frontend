# 数据库设计

数据库：MySQL 8.x | 库名：`robot_factory_cloud` | 字符集：utf8mb4_general_ci

## 现有核心表

### genie_external_agent（统一 Agent/Tool 注册表）

已在生产环境运行，24 条记录。

| 字段 | 类型 | 说明 |
|---|---|---|
| id | BIGINT PK | 自增主键 |
| agent_name | VARCHAR(64) UK | Agent 唯一标识 |
| display_name | VARCHAR(128) | 显示名称 |
| description | TEXT | Agent 描述（供 LLM 理解） |
| agent_type | VARCHAR(20) | 接入协议：`mcp` / `http_api` / `builtin` |
| mode | VARCHAR(20) | 运行模式：`TOOL` / `SUBAGENT` / `PAGE_APP` / `ALL` |
| display_template | VARCHAR(50) | 前端展示模板：file/image/audio/video/app/microService/search_web/search_file/answer/ai_answer |
| spec_json | LONGTEXT | 原始协议 spec（JSON），含 url/api_key/headers/timeout |
| allowed_tools | TEXT | 允许工具（逗号分隔） |
| denied_tools | TEXT | 禁止工具（逗号分隔） |
| system_prompt | TEXT | 系统提示词 |
| parameters_schema | TEXT | 工具参数 Schema（JSON） |
| max_concurrency | INT | 最大并发数 |
| max_steps | INT | 最大步数 |
| temperature | DECIMAL(3,2) | 生成温度 |
| runtime_role | VARCHAR(20) | 运行角色：tool / subagent / page_app |
| interaction_mode | VARCHAR(20) | 交互方式：sync / async / stream |
| dispatch_mode | VARCHAR(20) | 调度方式：tool_sync / subagent / page_app |
| quality_score | DOUBLE | 质量分（待迁移到指标表） |
| avg_latency_ms | INT | 平均时延 ms（待迁移） |
| success_rate | DOUBLE | 历史成功率（待迁移） |
| avg_token_cost | INT | 平均 Token 成本（待迁移） |
| call_count | BIGINT | 累计调用次数（待迁移） |
| corp_id | BIGINT | 租户 ID |
| client_code | VARCHAR(32) | 客户端标识 |
| is_public | TINYINT | 1=公共 0=需检查权限 |
| allowed_roles | VARCHAR(500) | 允许的权限组 ID |
| hidden | TINYINT | 是否隐藏 |
| sort_order | INT | 排序 |
| icon | VARCHAR(32) | 图标标识 |
| yn | TINYINT | 软删除 |

### genie_agent_permission（权限控制表）

| 字段 | 类型 | 说明 |
|---|---|---|
| id | BIGINT PK | 自增主键 |

## 计划新增/改造

### genie_external_agent 改造

```sql
ALTER TABLE genie_external_agent
  ADD COLUMN parent_id BIGINT COMMENT '父级ID，MCP Server下的Tool指向其Server',
  ADD COLUMN provider_id BIGINT COMMENT 'FK to genie_provider',
  ADD COLUMN category_id BIGINT COMMENT 'FK to genie_category',
  ADD COLUMN source_type VARCHAR(30) COMMENT 'platform_builtin/lowcode/workflow/code_agent/page_app/mcp_cloud/http_partner',
  ADD COLUMN status VARCHAR(20) DEFAULT 'published' COMMENT 'draft/testing/published/deprecated/disabled',
  ADD COLUMN version VARCHAR(20) DEFAULT '1.0.0';
```

### genie_provider（服务提供商）

```sql
CREATE TABLE genie_provider (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  provider_code VARCHAR(32) UNIQUE NOT NULL,
  provider_name VARCHAR(128) NOT NULL,
  provider_type VARCHAR(20) NOT NULL COMMENT 'internal/partner/cloud',
  description TEXT,
  logo VARCHAR(256),
  contact_name VARCHAR(64),
  contact_email VARCHAR(128),
  auth_type VARCHAR(20) DEFAULT 'api_key' COMMENT 'api_key/oauth2/cert/none',
  auth_config JSON,
  base_url VARCHAR(512),
  status VARCHAR(20) DEFAULT 'active',
  contract_expire_date DATE,
  yn TINYINT DEFAULT 1,
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### genie_category（分类体系）

```sql
CREATE TABLE genie_category (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  category_code VARCHAR(32) UNIQUE NOT NULL,
  category_name VARCHAR(64) NOT NULL,
  parent_id BIGINT DEFAULT NULL,
  icon VARCHAR(32),
  sort_order INT DEFAULT 0,
  yn TINYINT DEFAULT 1,
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

分类：校园业务（教务/图书馆/一卡通/OA）、教学科研、通用能力（文档生成/图像/搜索/代码）、数据分析、生活服务

### genie_agent_metrics（运行指标，从主表拆出）

```sql
CREATE TABLE genie_agent_metrics (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  agent_id BIGINT UNIQUE NOT NULL,
  quality_score DOUBLE DEFAULT 0.5,
  avg_latency_ms INT DEFAULT 0,
  success_rate DOUBLE DEFAULT 1.0,
  avg_token_cost INT DEFAULT 0,
  call_count BIGINT DEFAULT 0,
  last_call_time DATETIME,
  last_health_check DATETIME,
  health_status VARCHAR(20) DEFAULT 'unknown' COMMENT 'healthy/degraded/down/unknown',
  p99_latency_ms INT DEFAULT 0,
  error_count_24h INT DEFAULT 0,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### genie_health_config（健康检查配置）

```sql
CREATE TABLE genie_health_config (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  agent_id BIGINT UNIQUE NOT NULL,
  check_type VARCHAR(20) DEFAULT 'http' COMMENT 'http/tcp/mcp_ping',
  check_url VARCHAR(512),
  check_interval_sec INT DEFAULT 60,
  timeout_ms INT DEFAULT 5000,
  healthy_threshold INT DEFAULT 3,
  unhealthy_threshold INT DEFAULT 2,
  enabled TINYINT DEFAULT 1,
  yn TINYINT DEFAULT 1,
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### genie_circuit_breaker（熔断降级配置）

```sql
CREATE TABLE genie_circuit_breaker (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  agent_id BIGINT UNIQUE NOT NULL,
  failure_threshold INT DEFAULT 5,
  failure_window_sec INT DEFAULT 60,
  open_duration_sec INT DEFAULT 30,
  half_open_max_calls INT DEFAULT 3,
  fallback_agent_id BIGINT,
  fallback_message TEXT,
  enabled TINYINT DEFAULT 1,
  current_state VARCHAR(20) DEFAULT 'closed' COMMENT 'closed/open/half_open',
  last_state_change DATETIME,
  yn TINYINT DEFAULT 1,
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### genie_agent_version（版本管理）

```sql
CREATE TABLE genie_agent_version (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  agent_id BIGINT NOT NULL,
  version VARCHAR(20) NOT NULL,
  changelog TEXT,
  spec_json_snapshot LONGTEXT,
  status VARCHAR(20) DEFAULT 'draft' COMMENT 'draft/testing/released/rollback',
  released_at DATETIME,
  created_by VARCHAR(64),
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY uk_agent_version (agent_id, version)
);
```

### genie_access_quota（调用配额/限流）

```sql
CREATE TABLE genie_access_quota (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  agent_id BIGINT NOT NULL,
  scope_type VARCHAR(20) NOT NULL COMMENT 'global/corp/group/user',
  scope_id VARCHAR(64),
  daily_limit INT,
  monthly_limit INT,
  qps_limit INT,
  current_daily_count INT DEFAULT 0,
  current_monthly_count INT DEFAULT 0,
  yn TINYINT DEFAULT 1,
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  UNIQUE KEY uk_quota (agent_id, scope_type, scope_id)
);
```

### genie_agent_tag（标签）

```sql
CREATE TABLE genie_agent_tag (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  agent_id BIGINT NOT NULL,
  tag_name VARCHAR(32) NOT NULL,
  UNIQUE KEY uk_agent_tag (agent_id, tag_name),
  KEY idx_tag (tag_name)
);
```
